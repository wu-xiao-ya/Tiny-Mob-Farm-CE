## 中文

本次 `1.0.8-beta` 同时覆盖 1.12.2 社区版与 Forge 1.16.5 beta 版。

1.16.5 移植版已同步 1.12.2 社区版的核心功能进度，包括 CraftTweaker 自定义产物、输出满暂停、输出失败重试、掉落来源缓存、机械农场、邻居方块变化重新检测，以及农场方块/物品形态渲染修正。

1.12.2 版本继续保留本次俄语本地化修正，提升俄语玩家的界面可读性和翻译完整度。

感谢社区贡献者 Yaroslavik / MrKoteo 提供俄语翻译更新。

## English

This `1.0.8-beta` release now covers both the 1.12.2 community build and the Forge 1.16.5 beta build.

The 1.16.5 port is aligned with the 1.12.2 community feature set, including CraftTweaker custom drops, output-full pausing, output retry handling, drop-source caching, mechanical farms, neighbor-change output rechecks, and farm block/item rendering fixes.

The 1.12.2 build keeps the Russian localization fixes, improving UI readability and translation coverage for Russian-speaking players.

Thanks to community contributor Yaroslavik / MrKoteo for the Russian translation update.
